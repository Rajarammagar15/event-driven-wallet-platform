# Event Driven Wallet Platform

A distributed wallet platform built using Spring Boot, Kafka, Redis, PostgreSQL, and MongoDB.

## Current Features

- Create Wallet
- Credit Wallet
- Debit Wallet
- Get Balance
- Redis Cache
- Kafka Event Publishing
- Pessimistic Locking
